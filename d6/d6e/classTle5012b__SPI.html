Tle5012b_SPI.h - Library for Arduino for the TLE5012B angle sensor.

GMR-based angle sensor for angular position sensing in automotive applications This library includes the 3-wire SPI connection for the TLE5012B -E1000/E5000/E9000 sensor2GO kits
