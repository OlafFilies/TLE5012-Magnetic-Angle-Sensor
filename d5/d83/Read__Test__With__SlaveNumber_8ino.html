Read_Test_With_SlaveNumber

This example allows the control of multiple sensors on one SPI channel \detail This example program starts one to four sensors on one SPI line with different CS pins and returns the angle value together with the safety word and the status register SNR. Due to the selected slave number the SNR will be set to 0x00, 0x01, 0x10 or 0x11. When you fetch a value from the sensor with safety word on than the safety word register RESP will indicate the sensor who is responding. This should be the same than you have ask for. The loop function will than print out the individual sensor(s) angle values.
